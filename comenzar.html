Descubre Quickly.
El sistema de diseño creado con CSS Grid.

Con Quickly, puedes comenzar integrando fácilmente el CSS y JavaScript en tu proyecto a través de CDN. Esta opción te libera de la necesidad de realizar pasos de compilación, permitiéndote iniciar tu proyecto rápidamente. Así, puedes concentrarte en lo que realmente importa: desarrollar tu aplicación de manera eficiente y sin complicaciones.

Inicia rápidamente

Comienza integrando fácilmente el CSS y JavaScript listos para producción de Quickly a través de CDN. Con esta opción, evitas la necesidad de realizar pasos de compilación, lo que te permite comenzar rápidamente con tu proyecto y enfocarte en lo que realmente importa: desarrollar tu aplicación de manera eficiente y sin complicaciones.

Enlace al CDN

A continuación, te proporcionamos los enlaces principales de nuestra CDN para que los utilices. Hemos separado los archivos JavaScript en dos unidades: una para componentes y otra para formularios. Si solo necesitas utilizar el JavaScript estándar que incluye componentes, simplemente haz la llamada a quicklyComp.js. Si necesitas ambos, componentes y formularios, realiza las dos llamadas correspondientes a quicklyComp.js y quicklyForm.js.

Detalle CDN URL
CSS general https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/css/quickly.min.css
JS Componentes https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/js/quicklyComp.min.js
JS Formulario https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/js/quicklyForm.min.js

Implementación rápida y simple

Para comenzar, crea un nuevo archivo index.html en la carpeta raíz de tu proyecto.

Copia y pega este HTML inicial, el cual incluye todos los elementos necesarios para comenzar tu proyecto con Quickly. Recuerda siempre colocar tus estilos CSS después de los de Quickly para asegurar que tengan prioridad.

Recomendamos incluir Normalize a tu proyecto para evitar inconsistencias en los estilos entre navegadores.

                  
    <!DOCTYPE html>
    <html lang="es">
        <head>
            <meta charset="UTF-8">
            <!-- Coloca aquí tu ICO -->
            <link rel="icon" href="<!-- Aquí -->">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <!-- Normalize -->
            <link rel="stylesheet" href="https://necolas.github.io/normalize.css/8.0.1/normalize.css">
            <!-- CDN Quickly -->
            <!-- Llamada para CSS -->
            <link href="https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/css/quickly.min.css" rel="stylesheet">
            <!-- Llamada para Javascript de componentes -->
            <script src="https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/js/quicklyComp.min.js"></script>
            <!-- Llamada para Javascript de Formularios solo si es necesario -->
            <script src="https://cdn.jsdelivr.net/gh/QuicklyCSS/Quickly@main/dist/v-1/js/quicklyForm.min.js"></script>
            <!-- Coloca aquí tu CSS personalizado -->
            <link href="<!-- Aquí -->" rel="stylesheet">
            <title>Quickly Demo</title>
        </head>
        <body>   
            <h1>Hola, mundo :)!</h1>
        </body>
    </html>
