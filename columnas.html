Sistema de Columnas CSS Grid en Quickly

En Quickly, el sistema de columnas de CSS Grid ofrece una forma rápida y sencilla de crear diseños flexibles y visualmente atractivos. Con solo unas pocas líneas de código, puedes organizar tu contenido en columnas perfectamente alineadas y adaptadas a cualquier dispositivo. La facilidad de implementación de CSS Grid en Quickly permite a los desarrolladores crear interfaces sofisticadas con una mínima curva de aprendizaje, lo que agiliza el proceso de desarrollo y garantiza resultados profesionales en poco tiempo.

Opciones de Columnas

El sistema de columnas de Quickly se adapta a los tres puntos de interrupción predeterminados y a cualquier punto de interrupción que personalices. Los tres niveles de columna predeterminados son los siguientes:

Tipo Clase Descripción
Normal .col-n° Más de 992px · Para televisores, computadores de escritorio y tablets.
Mediano .col-m-n° Menos de 992px · Para teléfonos móviles o tabletas.
Pequeño .col-s-n° Menor a 576px · Para teléfonos móviles.

¿Y cómo se utiliza la clase .col?

La clase .col es esencial en el desarrollo web para crear diseños adaptables. Al aplicarla a elementos HTML, como divs, podemos dividir el espacio disponible en columnas proporcionales dentro de un contenedor principal. Esto facilita la creación de diseños responsivos que se ajustan automáticamente a diferentes tamaños de pantalla, garantizando una experiencia de usuario óptima en dispositivos móviles y de escritorio. A continuación, presentamos una demostración de cómo aplicar las clases .col.

Importante
.col: Se utiliza en elementos hijos de un contenedor de cuadrícula GRID para que el elemento abarque el espacio determinado en una columna (Item).

Clase Descripción
.col-2
(1,2,3,...12)		 Se utiliza para especificar cuántos espacios debe ocupar la columna en los item de la cuadrícula. En este caso específico, col-2 indica que el elemento debe ocupar dos espacios para la columna en la cuadrícula de 3 item (item-3).
