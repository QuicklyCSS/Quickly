Entradas de datos (Input Text)

Descubre la simplicidad y flexibilidad de Quickly para diseñar formularios con entradas de datos personalizadas. Aplica estilos y personalízalas fácilmente para que se adapten a tu diseño de manera intuitiva y atractiva.

Entrada tipo texto

Las entradas de tipo "text" con validación están diseñadas específicamente para aceptar únicamente datos de texto. Esto significa que cualquier otro tipo de dato, como números, fechas o formatos especiales, será rechazado automáticamente.

Atributo / Clase
Descripción
required
Para activar la validación de los campos, debes especificar el atributo.
disabled / .disabled
Para deshabilitar el campo, debes especificar el atributo correspondiente o la clase.

HTML 
   
    <!-- Requerido -->                                                                                                                                                             
    <div class="form-group">
        <label for="nombre" class="form-label">Tipo text</label>
        <input type="text" class="form-control" id="nombre" value="" placeholder="Ingresa tu nombre" name="nombre" required aria-required="true" aria-describedby="nombre-instruccion">
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                          
    </div>     
    
    <!-- No Requerido --> 
    <div class="form-group">
        <label for="nombre" class="form-label">Tipo text</label>
        <input type="text" class="form-control" id="nombre" value="" placeholder="Ingresa tu nombre" name="nombre" aria-required="true" aria-describedby="nombre-instruccion">
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                          
    </div>  

    <!-- Deshabilitado -->
    <div class="form-group">
        <label for="nombre" class="form-label">Tipo text</label>
        <input type="text" class="form-control disabled" id="nombre" value="" placeholder="Ingresa tu nombre" name="nombre" disabled aria-required="true" aria-describedby="nombre-instruccion">
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                          
    </div>
