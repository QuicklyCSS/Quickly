Opciones de Grilla

El sistema de cuadrícula de Quickly se adapta a los tres puntos de interrupción predeterminados y a cualquier punto de interrupción que personalices. Los tres niveles de cuadrícula predeterminados son los siguientes:

Tipo Clase Descripción
Normal .item-n° > 992px | Para televisores, computadores de escritorio y tablets.
Mediano .item-m-n° < 992px | Para teléfonos móviles o tabletas en orientación vertical.
Pequeño .item-s-n° < 576px | Para teléfonos móviles en modo vertical.

.Item-auto

Clase Descripción
.item-auto Establece un grid con un total de 12 item (columnas), donde todas tienen el mismo tamaño, cada una ocupando una fracción del espacio disponible.

.Item-auto-fill

Clase Descripción
.item-auto-fill Se utiliza para crear una cuadrícula donde las columnas se ajustan automáticamente para llenar el contenedor, con un ancho mínimo de 200px y un ancho máximo de 1fr. Las filas tienen una altura automática, ajustándose según el contenido.

.Item-auto-fit

Clase Descripción
.item-auto-fit Crea una cuadrícula con un número dinámico de columnas que se ajustarán automáticamente para llenar el contenedor, con cada columna teniendo un ancho mínimo de 200px y expandiéndose para llenar el espacio disponible. Si el espacio disponible es insuficiente para todas las columnas, algunas de ellas se ajustarán automáticamente para caber en la pantalla.
