Botones

Explore la variedad de estilos de botones personalizados de Quickly para diversas acciones y funcionalidades en formularios y diálogos, con soporte para tamaños y estados diversos.

Botón básico

Esta es la forma más básica de definir un elemento como botón y asignarle propiedades. Es un estilo sencillo que puede ser personalizado y expandido según las necesidades específicas de diseño.

.btn

Clase Descripción
.btn La clase define un botón con un tamaño de relleno, bordes redondeados, cursor interactivo, sin borde visible y un fondo transparente.
.btn-ico Botón de Icono: Representa la versión exclusiva de un botón en formato SVG diseñado para alojar un icono.

Botón base

HTML 
                                                                        
    <button type="button" class="btn">Botón de Ejemplo Estándar</button>
    <button type="button" class="btn-ico"><svg class="ico ico-dark".../></svg></button>

Opciones de tamaños

Puedes optar por tres tamaños de botones: el estándar, que es el predeterminado, junto con el pequeño y el grande. Cambiar entre ellos es fácil, solo necesitas sustituir la clase ".btn" por la que desees utilizar.

.btn-(opción)

Clase Descripción
.btn-s .Boton-Small. Define el tamaño más reducido de un botón, disminuyendo tanto el tamaño del texto como las proporciones xy.
.btn .Boton Estandar. Representa la versión predeterminada del botón.
.btn-l .Boton-Large. Corresponde a la versión ampliada del botón, que aumenta tanto el tamaño del texto como las proporciones xy.

Botón base

HTML 
                                    
    <button type="button" class="btn-s btn-primary">Botón Pequeño</button>                          
    <button type="button" class="btn btn-primary">Botón Normal</button>                            
    <button type="button" class="btn-l btn-primary">Botón Grande</button>

Botones en etiquetas

Las clases .btn están diseñadas principalmente para ser utilizadas con la etiqueta <button>, pero también pueden aplicarse a elementos <a> o <input> (La representación visual puede variar en algunos navegadores).
Cuando emplees clases de botón en elementos <a> destinados a activar funciones dentro de la página, como la expansión de contenido, en lugar de enlazar con nuevas páginas o secciones dentro de la misma página, asegúrate de etiquetar estos enlaces con role="button". Esto garantizará una comunicación efectiva de su propósito a las tecnologías de asistencia, como los lectores de pantalla.

Etiqueta Descripción
a Para enlazar un botón sin usar el atributo onclick en Bootstrap con la etiqueta <button>, puedes simplemente envolver el botón dentro de una etiqueta <a> y definir el enlace en el atributo "href" de la etiqueta <a>
Button JS Accion mediante JavaScript, cambia la URL actual a '#' mediante JavaScript.
Button Submit Puedes representar un botón de envío (submit) con el estilo btn btn-primary.
input Esta etiqueta define un botón con clase btn y btn-primary.
input Submit Crea un botón de envío (submit) con el estilo btn btn-primary.
input Reset Crea un botón de restablecimiento con el estilo btn btn-primary

Botón usos en etiquetas

Enlace
HTML 
                                    
    <a class="btn btn-primary" href="#" role="button">Enlace</a>
    <button type="button" class="btn btn-primary" onclick="window.location.href='#'">Button JS</button>
    <button class="btn btn-primary" type="submit">Button Submit</button>
    <input class="btn btn-primary" type="button" value="Input">
    <input class="btn btn-primary" type="submit" value="Submit">
    <input class="btn btn-primary" type="reset" value="Reset">
