Uso de Alineaciones en Quickly

¡Desata tu creatividad con las alineaciones! Desde las opciones más simples hasta las avanzadas como space-around y stretch, CSS Grid te brinda un abanico de posibilidades para diseñar interfaces web únicas y atractivas. Descubre cómo las alineaciones en CSS Grid pueden elevar tus diseños al siguiente nivel y cautivar a tu audiencia desde el primer vistazo.

Alineaciones Simples y Efectivas

En Quickly, implementar alineaciones para textos y objetos es rápido y sencillo. Con opciones para alinear textos a la izquierda, derecha o centrados, así como para posicionar objetos con precisión, podrás crear diseños visualmente atractivos en pocos pasos. Aprovecha nuestras herramientas intuitivas para lograr la alineación perfecta y destacar tu contenido de manera efectiva.

Recuerda siempre anteponer la clase .grid al utilizar las clases de alineación, es un detalle importante a tener en cuenta.

Alineación center center

Para centrar un elemento en el medio del contenedor, utiliza la clase "align-cc".

.align-cc

Clase Descripción
.align-cc Align-center-center. Alinea los elementos verticalmente y orizontalmente al centro del contenedor.

Alineación centro centro

align-cc
HTML 
                                    
    <div class="grid align-cc">
        <div class="p-2 text-ac">item 1</div>
    </div>

.align-vc Align-vertical-center. Alinea los elementos al centro verticalmente dentro del contenedor.
.align-ve Align-vertical-end. Alinea los elementos verticalmente en el extremo derecho del contenedor.

Control estiramiento horizontal y vertical en contenedores

Por defecto, los elementos dentro de un contenedor GRID se expanden para ocupar todo el ancho y alto disponibles. Sin embargo, en ocasiones no queremos que los elementos se expandan horizontal o verticalmente al aplicar la clase "grid". En estos casos, podemos usar las clases "no-stretch", que evitan la expansión vertical.

.no-stretch

Clase Descripción
.no-stretch No-Stretch. Evita que los elementos se expandan vertical y horizontalmente.
.no-stretch-h No-Stretch-Horizontal. Previene la expansión horizontal de los elementos en el contenedor.
.no-stretch-v No-Stretch-Vertical. Previene la expansión vertical de los elementos en el contenedor.

Estiramiento horizontal y vertical

no-stretch

no-stretch-h

no-stretch-v

HTML 
                                    
    <div class="grid item-3 g-2 h-150 w-600">                                     
        <p class="bg-white border rounded p-2 no-stretch">no-stretch</p>   
        <p class="bg-white border rounded p-2 no-stretch-h">no-stretch-h</p>      
        <p class="bg-white border rounded p-2 no-stretch-v">no-stretch-v</p>                             
    </div>

.space-around

Recuerda siempre anteponer la clase .grid al utilizar las clases de espacios, es un detalle importante a tener en cuenta.

Clase Descripción
.space-ha Space-Horizontal-Around. Se distribuyen uniformemente a lo largo del eje horizontal con espacio alrededor de cada elemento.
.space-va Space-Vertical-Around. Se distribuyen uniformemente a lo largo del eje vertical con espacio alrededor de cada elemento.

Espacios no iguales al inicio y fin, pero entre los elementos son iguales horizontal y vertical
