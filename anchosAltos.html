Anchos y Altos

Con Quickly, puedes asignar alturas y anchos predefinidos a los elementos según tus preferencias o diseño. Además, disponemos de clases especiales para la versión móvil, lo que permite un mejor control del diseño.

Anchos en px

Asigna anchos exactos a tus diseños utilizando las clases "w".

Clase Descripción
.w-100 Establece un ancho fijo de 100px para el elemento.
.w-150 Establece un ancho fijo de 150px para el elemento.
.w-200 Establece un ancho fijo de 200px para el elemento.
.w-250 Establece un ancho fijo de 250px para el elemento.
.w-300 Establece un ancho fijo de 300px para el elemento.
.w-350 Establece un ancho fijo de 350px para el elemento.
.w-400 Establece un ancho fijo de 400px para el elemento.
.w-500 Establece un ancho fijo de 500px para el elemento.
.w-550 Establece un ancho fijo de 550px para el elemento.
.w-600 Establece un ancho fijo de 600px para el elemento.
.w-auto El ancho se ajustará automáticamente para que coincida con el ancho de su contenido.

Ancho y alto en %.

Puedes asignar alturas y anchos predefinidos a los elementos utilizando porcentajes. Además, puedes gestionar y adaptar mejor el diseño para las versiones móviles, donde las clases específicas para dispositivos móviles tienen prioridad sobre las clases normales, las cuales se desactivan.

.wp-90 .wpm-90 El ancho será el 90% del ancho del contenedor padre.
.wp-100 .wpm-100 El ancho será el 100% del ancho del contenedor padrer.
