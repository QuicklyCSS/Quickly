Opacidad

Controla la transparencia de los elementos utilizando las clases de opacidad con Quickly. Estas clases permiten ajustar el nivel de transparencia de cualquier elemento en tu diseño, proporcionando flexibilidad y mejorando la estética visual.

.opacity-n°

Clase Descripción
.opacity-10 Este elemento presentará una transparencia del 10% respecto a su color original.
.opacity-20 Este elemento presentará una transparencia del 20% respecto a su color original.
.opacity-30 Este elemento presentará una transparencia del 30% respecto a su color original.
.opacity-40 Este elemento presentará una transparencia del 40% respecto a su color original.
.opacity-50 Este elemento presentará una transparencia del 50% respecto a su color original.
.opacity-60 Este elemento presentará una transparencia del 60% respecto a su color original.
.opacity-70 Este elemento presentará una transparencia del 70% respecto a su color original.
.opacity-80 Este elemento presentará una transparencia del 80% respecto a su color original.
.opacity-90 Este elemento presentará una transparencia del 90% respecto a su color original.
.opacity-100 Este elemento presentará una transparencia del 100% respecto a su color original.
