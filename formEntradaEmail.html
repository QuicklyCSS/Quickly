Entradas de datos (Input Email)

Descubre la simplicidad y flexibilidad de Quickly para diseñar formularios con entradas de tipo email personalizadas. Aplica estilos y personalízalas fácilmente para que se adapten a tu diseño de manera intuitiva y atractiva.

Entrada tipo Email

Las entradas de tipo "text" con validación están diseñadas específicamente para aceptar únicamente datos de tipo email.

Atributo / Clase
Descripción
required
Para activar la validación de los campos, debes especificar el atributo.
disabled / .disabled
Para deshabilitar el campo, debes especificar el atributo correspondiente o la clase.
