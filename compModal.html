Modal

Puedes utilizar los modales de Quickly para incorporar ventanas emergentes en tu sitio web, ya sea para mostrar información destacada, alertas importantes para los usuarios o incluso para presentar contenido altamente personalizado de manera interactiva. ademas el contenido de las modales en el .modal-body tiene aplicado un alto maximo de 300px

Modales

Para implementar un modal, es esencial incluir en el código las etiquetas de atributo personalizado data-Qy-toggle="modal", junto con el identificador id="modal". Si planeas incorporar múltiples modales en tu sitio web o aplicación, necesitas agregar tanto el atributo personalizado como el ID con el mismo nombre y una secuencia numérica, por ejemplo, modal-1, modal-2, modal-3, etc., tal como se ilustra en los ejemplos a continuación.
