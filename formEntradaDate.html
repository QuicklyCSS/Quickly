Entradas de datos (Input Date)

Descubre la simplicidad y flexibilidad de Quickly para diseñar formularios con entradas de tipo fecha personalizadas. Aplica estilos y personalízalas fácilmente para que se adapten a tu diseño de manera intuitiva y atractiva.

Entrada Tipo Fecha

Las entradas de tipo "date" con validación están diseñadas específicamente para aceptar únicamente datos de fecha.

Atributo / Clase Descripción
required Para activar la validación de los campos, debes especificar el atributo.
disabled / .disabled Para deshabilitar el campo, debes especificar el atributo correspondiente o la clase.

HTML 
   
    <!-- Requerido -->                                                                                                                                                             
    <div class="form-group">
        <label for="fecha" class="form-label">Fecha</label>
        <input type="date" class="form-control" id="fecha" name="fecha" required>
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                          
    </div>     
    
    <!-- No Requerido --> 
    <div class="form-group">
        <label for="fecha" class="form-label">Fecha</label>
        <input type="date" class="form-control" id="fecha" name="fecha">
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                            
    </div>  

    <!-- Deshabilitado -->
    <div class="form-group">
        <label for="fecha" class="form-label">Fecha</label>
        <input type="date" class="form-control" id="fecha" name="fecha" disabled>
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>                                            
    </div>
