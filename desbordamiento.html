Desbordamiento

Utiliza las clases de desbordamiento para controlar eficientemente el contenido de un elemento.

Tipos de desbordamientos

Para controlar el desbordamiento de los elementos dentro de un contenedor puedes utilizar las clases .scroll.

.scroll-opción (a-h-x-y)

Clase
Descripción
.scroll-h
Scroll-Hidden. Ocultara cualquier contenido que se desborde del área asignada del elemento.
.scroll-a
Scroll-Auto. Las barras de desplazamiento vertical u horizontal se mostrarán automáticamente si el contenido excede los límites del elemento. Si el contenido no se desborda, las barras de desplazamiento no serán visibles.
.scroll-x
Scroll-X. Proporcionará barras de desplazamiento horizontal aunque su contenido no exceda su tamaño definido.
.scroll-y
Scroll-Y. Proporcionará barras de desplazamiento vertical aunque su contenido no exceda su tamaño definido.
