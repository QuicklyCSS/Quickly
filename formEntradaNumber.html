Entradas de datos (Input Number)

Descubre la simplicidad y flexibilidad de Quickly para diseñar formularios con entradas de tipo número personalizadas. Aplica estilos y personalízalas fácilmente para que se adapten a tu diseño de manera intuitiva y atractiva. Nuestra biblioteca de estilos garantiza que los campos numéricos sean visualmente consistentes y funcionalmente precisos, facilitando la entrada de datos numéricos y mejorando la experiencia del usuario.

Entrada Tipo Número

Las entradas de tipo "number" con validación están diseñadas específicamente para aceptar únicamente datos de números. Esto significa que cualquier otro tipo de dato, será rechazado automáticamente.

Atributo / Clase
Descripción
required
Para activar la validación de los campos, debes especificar el atributo.
disabled / .disabled
Para deshabilitar el campo, debes especificar el atributo correspondiente o la clase.

HTML 
     
    <!-- Requerido -->  
    <div class="form-group">
        <label for="steps" class="form-label">Número (Requerido)</label>
        <input type="number" id="steps" name="steps" min="1" max="" step="1" aria-valuemin="1" required aria-valuemax="">
        <div class="valid-feedback">¡Texto ok!</div>
        <div class="invalid-feedback">¡Texto error!</div>      
    </div>        
    
    <!-- No Requerido --> 
    <div class="form-group">
        <label for="steps" class="form-label">Número (No Requerido)</label>
        <input type="number" id="steps" name="steps" min="1" max="" step="1" aria-valuemin="1" aria-valuemax="">
        <div class="valid-feedback">¡Muy bien!</div>
        <div class="invalid-feedback">¡Por favor, debes ingresar un número!</div>
    </div>    
    
    <!-- Deshabilitado -->    
    <div class="form-group">
        <label for="steps" class="form-label">Número (Deshabilidato)</label>
        <input class="disabled" type="number" id="steps" name="steps" min="1" max="" step="1" aria-valuemin="1" aria-valuemax="" disabled>
        <div class="valid-feedback">¡Muy bien!</div>
        <div class="invalid-feedback">¡Por favor, debes ingresar un número!</div>
    </div>
