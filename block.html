Uso de Alineaciones en Quickly

¡Desata tu creatividad con las alineaciones! Desde las opciones más simples hasta las avanzadas como space-around y stretch, CSS Grid te brinda un abanico de posibilidades para diseñar interfaces web únicas y atractivas. Descubre cómo las alineaciones en CSS Grid pueden elevar tus diseños al siguiente nivel y cautivar a tu audiencia desde el primer vistazo.

Alineaciones Simples y Efectivas

En Quickly, implementar alineaciones para textos y objetos es rápido y sencillo. Con opciones para alinear textos a la izquierda, derecha o centrados, así como para posicionar objetos con precisión, podrás crear diseños visualmente atractivos en pocos pasos. Aprovecha nuestras herramientas intuitivas para lograr la alineación perfecta y destacar tu contenido de manera efectiva.

Control vertical en contenedores

Por defecto, los elementos dentro de un contenedor se expanden horizontalmente para ocupar todo el ancho disponible. Sin embargo, para lograr que se expandan verticalmente, necesitarás aplicar la clase "stretch". Esta clase fuerza al elemento a estirarse verticalmente dentro de su contenedor.

.stretch

Clase
Descripción
.stretch
Stretch. Los elementos se estiran verticalmente para ocupar todo el ancho y alto disponible.

Estiramiento horizontal y vertical

Defecto
Stretch
HTML 
                                    
    &lt;div class="m-5 grid item-auto-fit g-5 h-200"&gt;
        &lt;div class="div1"&gt;                                                                      
            &lt;div class="div2 p-2 text-c"&gt;Defecto&lt;/div&gt;                                          
        &lt;/div&gt;   
        &lt;div class="div1 stretch"&gt;                                                              
            &lt;div class="div2 p-2 text-c align-vc"&gt;Stretch&lt;/div&gt;                        
        &lt;/div&gt;                          
    &lt;/div&gt;

Control vertical en contenedores

A veces, no queremos que los elementos se expandan verticalmente al aplicar la clase "grid". En tales casos, podemos emplear las clases "no-stretch", las cuales previenen la expansión vertical.

.stretch

Clase
Descripción
.no-stretch
No-Stretch. Evita que los elementos se expandan verticalmente.
.no-stretch-vs
No-Stretch-Vertical-Start. Previene la expansión vertical de los elementos al posicionarlos en la parte superior del contenedor.
.no-stretch-vc
No-Stretch-Vertical-Center. Previene la expansión vertical de los elementos al posicionarlos en la parte central del contenedor.
.no-stretch-ve
No-Stretch-Vertical-End. Previene la expansión vertical de los elementos al posicionarlos en la parte inferior del contenedor.
.no-stretch-h
No-Stretch-Horizontal. Previene la expansión horizontal de los elementos en el contenedor.
.no-stretch-v
No-Stretch-Vertical. Previene la expansión vertical de los elementos en el contenedor.
