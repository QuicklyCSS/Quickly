Iconos

Quickly no incluye una biblioteca de iconos incorporada, dejando esta elección abierta para tu proyecto. Sin embargo, proporcionamos clases que facilitan la incorporación de iconos en botones, botones de estilo fantasma y botones de ícono.

Tamaños de iconos

Ofrecemos seis opciones de tamaño para que puedas personalizar el tamaño de tus iconos según tus preferencias.

Para que surtan efecto, estas clases .ico deben aplicarse a las imágenes vectoriales SVG, complementadas con las clases de colores .ico-primary. A continuación, te mostramos cómo hacerlo.

.ico

Clase
Descripción
.ico
Clase por defecto para ser aplicada en botones, establece la altura del icono en 12 píxeles.
.ico-8
Establece la altura del icono en 8 píxeles.
.ico-16
Establece la altura del icono en 16 píxeles.
.ico-24
Establece la altura del icono en 24 píxeles.
.ico-32
Establece la altura del icono en 32 píxeles.
.ico-40
Establece la altura del icono en 40 píxeles.

Colores aplicados a los iconos

Puedes aplicar color a los iconos utilizando las clases .ico-primary o las otras clases de color que mencionaremos a continuación.

.ico-(opción)

Clase
Descripción
.ico-white
Define el color blanco para el icono.
.ico-primary
Define el color primario 1 para el icono.
.ico-primary-alt
Define el color primario 2 (alternativo) para el icono.
.ico-secondary
Define el color secundario 1 para el icono.
.ico-secondary-alt
Define el color secundario 2 (alternativo) para el icono.
.ico-success
Define el color de éxito para el icono.
.ico-danger
Define el color de peligro para el icono.
.ico-warning
Define el color de advertencia para el icono.
.ico-info
Define el color de información para el icono.
.ico-light
Define el color claro para el icono.
.ico-dark
Define el color oscuro para el icono.
.ico-link
Define el color de enlace para el icono.
