Template grid en Quickly

Descubre nuestra sección de Template Grid, diseñada para ofrecer soluciones rápidas y eficientes a tus proyectos web. Con una variedad de layouts preconfigurados, los templates te permiten crear estructuras flexibles y responsivas con CSS Grid, adaptándose a cualquier tipo de sitio o aplicación. Desde distribuciones simples hasta diseños más complejos, estos ejemplos están pensados para facilitar el desarrollo, optimizando tanto el tiempo como la experiencia de usuario.

Grilla de plantilla

Para aplicar correctamente las clases de la cuadrícula (template grid), es fundamental que el contenido esté estructurado con identificadores (`id`) adecuados, como `header`, `nav`, `main`, `aside` y `footer`. Estos identificadores proporcionan una estructura clara y semántica para la página. Al envolver el contenido con ellos, la cuadrícula definirá cómo se distribuyen y organizan visualmente estas secciones dentro del layout, asegurando una presentación coherente, accesible y visualmente atractiva.

.temp-g-1

Clase Descripción
.temp-g-1 Versión Escritorio. La clase organiza una cuadrícula de dos columnas y cuatro filas, con el header ocupando toda la primera fila. En la segunda fila el Nav, en la terecera fila el main está en la primera columna ajustándose al espacio disponible y el aside en la segunda con un ancho max de 250px. La tercera fila repite el aside en la primera columna y coloca el footer en la segunda, creando un layout con barra lateral fija y contenido flexible.
Versión Móvil. La clase organiza todo el contenido en una sola columna, con el header ocupando la parte superior, seguido por el aside, luego el main, y finalmente el footer. Cada sección se apila verticalmente, ocupando el 100% del ancho de la pantalla, ajustando las alturas automáticamente según el contenido para una mejor visualización en dispositivos móviles.

Establece una distribución de 2 columnas y 4 filas.

    </div>

.temp-g-4

Clase Descripción
.temp-g-4 Define un diseño de cuadrícula ideal para páginas como formularios de inicio de sesión u otras páginas similares. La configuración asegura que el área main ocupe el alto completo disponible entre header y footer, proporcionando una estructura clara y centrada que se adapta bien a páginas con un enfoque en contenido central y acciones específicas.

.temp-g-5

Clase Descripción
.temp-g-5 Versión Escritorio. Se define un layout de dos columnas y tres filas usando CSS Grid, con una primera columna flexible de hasta 250px y una segunda que ocupa el espacio restante. El header abarca el ancho completo en la primera fila, mientras que el nav ocupa la primera columna en las filas dos y tres, funcionando como una barra lateral. El main se sitúa en la segunda columna de la segunda fila, y el footer se extiende en la tercera fila a lo largo de ambas columnas. Es un diseño ideal para páginas con navegación lateral y contenido flexible
Versión Móvil. Ajusta el layout a una sola columna, distribuyendo las áreas en filas consecutivas para optimizar la visualización en pantallas pequeñas. El header, nav, main, y footer se apilan verticalmente, ocupando cada uno el ancho completo del contenedor. Este diseño simplifica la estructura, eliminando el aside y asegurando una disposición clara y fácil de navegar, ideal para mejorar la experiencia del usuario en dispositivos móviles.
